# Encrava

This project is built with **Next.js** and **Tailwind CSS**, making it fast, responsive, and easy to extend.

---

## Tech Stack

- Next.js
- Tailwind CSS
- TypeScript (if applicable)

---

## Getting Started

Follow these steps to get Encrava running on your local machine.

---

## 📁 1. Clone the repository

```bash
git clone https://github.com/Kileyydev/Encrava

Then navigate into the project folder:

cd encrava

## 2. Install dependencies

Make sure you have Node.js (LTS version recommended) installed.

Install all required packages:

npm install

 3. Set up environment variables

Create a .env.local file in the root of the project:

touch .env.local

Then open it and add your required variables:

NEXT_PUBLIC_API_URL=your_api_url_here
# Add any other keys required by the project
 4. Run the development server

Start the project locally:

npm run dev

Then open your browser and go to:

http://localhost:3000
